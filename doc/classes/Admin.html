Admin

Definisce l'amministratore del database remoto.

Per poter registrare un nuovo utente sul database è necessario operare come amministratore quindi bisogna fornire le credenziali di amministrazione.

Per istanziare un nuovo amministratore bisogna definire prima un oggetto adminConf che specifichi tutte le sue informazioni; tale oggetto deve avere le seguenti properties:

  • name - string: username dell'amministratore
  • password - string: password dell'amministratore

Esempio:

var adminConf = {
    name: "admin",
    password: "1234"
};

var admin = new Admin( adminConf );
